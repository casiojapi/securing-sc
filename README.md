## audit process

+ clarification before report (sec -> dev)
+ receive report (sec -> dev)
+ clarification after report (dev -> sec)
+ address issues (dev)
+ local audit (dev)
+ audit follow up [request audit on new diff] (dev -> sec)
+ iterate
